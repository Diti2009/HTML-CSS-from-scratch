This is an example for Input tag HTML

Sample Code: 

     <input type="" name="" placeholder="" checked/>

Types of input tags

  • text - text input, default value of input tag is text
  • checkbox - checkboxes, allows multiple selection
  • radio - similar to checkbox, supports both multi and single selection
  • range - range slider
  • button - clickable button
  • file - allows file upload
  • password - obscured text input
  • date - text input with calendar to select dates
  • color - color picker

Attributes of Input Tags

  • placeholder- hint for the value to be entered
  • name - a unique name, radio input require same name attribute to be specified in input fields that belong to the same group of inputs to allow single selection
  • value - specifies the text content of input field, used for text and button type inputs
  • checked - a binary attribute that specifies the input fields that are selected, used for checkbox and radio type inputs
