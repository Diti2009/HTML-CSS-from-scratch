This is an example for Tables tag HTML

Sample Code: 

     <table bgcolor=color>
    	 <tr bgcolor=color>
    	 <th bgcolor=color>table header</th>
    	 </tr>

    	 <tr>
    	 <td bgcolor=color>table definiton</td>
    	 </tr>
     </table>

Attributes of table tags

  • bgcolor - specifies background color
  • eg- bgcolor=red or black or #ffffff
