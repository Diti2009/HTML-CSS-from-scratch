This is an example for Tables tag HTML

Sample Code: 

     <table>
    	 <tr>
    	 <th colspan="2">table header</th>
    	 </tr>

    	 <tr>
    	 <td rowspan="2">table definiton</td>
    	 </tr>
     </table>

Attributes of table tags

  • rowspan - specifies row span
  • colspan - specifies column span
  • eg- rowspan=2...10..
  • eg- colspan=2...10..
