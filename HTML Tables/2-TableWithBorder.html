This is an example for Tables tag HTML

Sample Code: 

     <table border=1>
    	 <tr>
    	 <th>table header</th>
    	 </tr>

    	 <tr>
    	 <td>table definiton</td>
    	 </tr>
     </table>

Attributes of table tags

  • border - specifies a border around the table, border width depends on the number specified
  • eg- border=1,2,3..10...
