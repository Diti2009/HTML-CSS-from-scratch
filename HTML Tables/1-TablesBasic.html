This is an example for Tables tag HTML

Sample Code: 

     <table>
    	 <tr>
    	 <th>table header</th>
    	 </tr>

    	 <tr>
    	 <td>table definiton</td>
    	 </tr>
     </table>

Overview of table tags

  • table - specifies a table
  • tr - specifies each row inside a table
  • th - specifies Table Header
  • td - specifies Table definition or Table data
